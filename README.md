# 🛒 Modern E-Commerce Website

A modern and responsive eCommerce website built using HTML, CSS, JavaScript, and Bootstrap. The project provides a smooth shopping experience with dynamic product rendering, product details, cart management, and wishlist functionality.

## Features

* Responsive eCommerce layout
* Product listing page
* Product detail page
* Shopping cart functionality
* Wishlist management
* Dynamic product rendering
* Local Storage integration
* Modern and clean UI design

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Bootstrap 5
* Local Storage API
